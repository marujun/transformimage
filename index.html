使用 transform进行div翻转图片或文字

Download this project as a .zip file Download this project as a tar.gz file

让元素支持3D变换需要的2个属性：perspective、transform-style

perspective(透视)：该属性会设置查看者的位置，并将可视内容映射到一个视锥上，继而投影到一个 2D 视平面上。如果不指定透视，则 Z 空间中的所有点将平铺到同一个 2D 视平面中，并且变换结果中将不存在景深概念。简单一点理解就是，perspective的值模拟的是用户观看动画时与动画中心面的焦距，值越小，看到的效果越明显()，越大，动画效果越不明显。当perspective为0的时候，感觉就像是你的眼睛紧贴着动画中心面看到的效果(看到的是平面动画)

transform-style(变换模式)：preserve-3d dom结构如下：

<div class="p" id="p" onclick="show();"> <div class="a"><img src="images/YHY010.jpg" class="image"></div> <div class="b"><img src="images/YHY011.jpg" class="image"></div> </div> 希望达到的效果是默认情况下，正面显示图片1，反面显示图片2，并且反面不可见。翻转180°后，图片2变为可见，图片1不可见，整个翻转的过程中，对于默认情况：

因为要做翻转，所以首先我们需要让a、b重合(绝对定位)，然后将b翻转180°，与a背靠背，这样同时翻转180°的时候，能始终保持一张图片可见。代码如下：

.p {-webkit-perspective:700;-webkit-transform-style: preserve-3d;} .a {-webkit-transform:rotateY(0deg);} .b {-webkit-transform: rotateY(-180deg);}

翻转时，让a、b同时顺时针翻转180°，代码如下：

.act .a {-webkit-transform:rotateY(180deg);} .act .b {-webkit-transform:rotateY(0deg);}

最后，设置a、b翻转动画时间，翻转圆点，以及设置翻转180°后背面是否可见，代码如下：

.a, .b { -webkit-transition:1.5s; -webkit-transform-origin: 50% 0; -webkit-backface-visibility:hidden; }
